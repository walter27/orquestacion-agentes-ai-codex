# Product Spec: Hello TODO

## Descripcion

Aplicacion TODO simple para gestionar tareas personales desde el navegador.

## Funcionalidades

F1. Ver interfaz inicial.
F2. Agregar tarea no vacia.
F3. Evitar tarea vacia o con solo espacios.
F4. Marcar tarea como completada.
F5. Eliminar tarea.
F6. Ver contador de pendientes.
F7. Mantener tareas luego de recargar usando localStorage.

## Fuera de alcance

- Login.
- Backend.
- Base de datos.
- Multiusuario.
- Sincronizacion en nube.
